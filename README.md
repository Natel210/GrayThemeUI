# GrayThemeUI
WPF resource DLL for simple themes.
You can use it as basic WPF.
Example samples are included.

## items
> - Default environment settings ( windows, page, user controll ... )
> - Square Button
> - Square ToggleButton
> - Check Box
> - Combo Box
> - Data Grid (-ing)
> - Group Box
> - Lebel
> - Separator Label
> - List Box
> - Numeric UpDown
> - TextBox
